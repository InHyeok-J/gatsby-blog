## 사용한 기술 스택

-   메인 : Gatsby
-   스타일 : styled-component
-   타이핑 이팩트 : typing-effect-reactjs
-   ...

## 기능

-   페이지 첫 로딩 시 타이핑 이펙트 ( sessionStorage를 활용한 탭마다 상태 저장) [구현]
-   네비게이션바 (home, search, tags, about, portfolio)[ 반응형 미구현 ]
-   인피니티 스크롤링이 적용된 메인 홈 페이지 [ 인피니팅 스크롤링 미구현 ]
-   ...
