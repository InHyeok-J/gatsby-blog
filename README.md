  Gatsby를 이용한 블로그
</h1>

## 사용한 기술 스택

- 메인 : Gatsby
- 스타일 : styled-component
- 타이핑 이팩트 : typing-effect-reactjs
- ...

## 구현

- 페이지 첫 로딩 시 타이핑 이펙트 ( sessionStorage를 활용한 탭마다 상태 저장)
- 네비게이션바 (home, search, tags, about)
- 메인페이지 반응형
- 포스트 페이지 마크다운 스타일링(커스텀 가능, 코드 하이라이팅)
- 포스트 페이지 TOC
- 포스트 페이지 깃헙 댓글 연동
- 포스트 페이지 TOC 스크롤별 색변화
- react-helmet
- favicon 설정
- 포스트 페이지 이전글 다음글
